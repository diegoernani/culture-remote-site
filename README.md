# Culture Remote — Site (Preview)
Stack: React + Vite + Tailwind + React Router

## Scripts
- `npm i`
- `npm run dev`  # abre o preview
- `npm run build` e `npm run preview`

## Rotas
- `/` — Landing Page
- `/basebook` — Página completa do Basebook v0.1

Editing:
- Altere cores em `tailwind.config.js` (petroleum, coral)
- Componentes em `src/components/*`
- Páginas em `src/pages/*`
